# Minesweeper-Game
MineSweeper Game with Java Swing

This game has 3 difficulity.
1. **Easy** : 8\*8 Board. 12 Mines.
2. **Normal**: 12\*12 Board. 23 Mines.
3. **Hard**: 19\*19 Board. 55 Mines.
